Tags used in HTML.

HTML has various tags through which we can code. some of these codes are:

  • p : This tag defines the paragraphs used in html coding.

  • a : This tag enables users to add links to their codes.

  • b : This tag can be used to make the text inside bold.

  • i : This tag can be used to make the text inside italic.

  • u : This tag is used to underline text that is typed inside it.

  • br : This tag can be used to add breaks in between paragraphs or text.

  • ahref : This tag is used for anchor links in an html code.

  • h1-h6 : These tags are used for headings the highest being h1 and the lowest being h6. Its currently using heading 3
